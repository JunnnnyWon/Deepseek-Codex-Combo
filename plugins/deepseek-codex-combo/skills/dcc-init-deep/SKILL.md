# dcc-init-deep

Inspect the current project, summarize durable conventions, and generate DCC-friendly project
memory without exposing hidden reasoning.

Generated `AGENTS.md` and `.dcc/*` outputs should be rerunnable: refresh the managed sections,
keep user-authored unmanaged notes intact, and write scan uncertainty explicitly when command
discovery is incomplete.
